1//! # JOSE Header Builder
2//!
3//! A Rust library for building JSON Web Signature (JWS) and JSON Web Encryption (JWE) headers
4//! based on the IANA JOSE registry specifications.
5//!
6//! This library provides type-safe builders for constructing JOSE headers with proper validation
7//! and serialization support.
8//!
9//! ## Features
10//!
11//! - Type-safe header construction
12//! - Support for all IANA-registered JOSE header parameters
13//! - Separate builders for JWS and JWE headers
14//! - Validation of header parameter combinations
15//! - Serde serialization/deserialization support
16//!
17//! ## Example
18//!
19//! ```rust
20//! use jose::{JwsHeaderBuilder, Algorithm};
21//!
22//! let header = JwsHeaderBuilder::new()
23//!     .algorithm(Algorithm::RS256)
24//!     .key_id("key-1")
25//!     .typ("JWT")
26//!     .build()
27//!     .unwrap();
28//!
29//! let json = serde_json::to_string(&header).unwrap();
30//! ```
